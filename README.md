# backendChuko
>> Aplicación para gestión de gastos del dia a dia
El plan de la aplicacion es tener:
- Gestion de usuarios
    -Usuario admin, gestor de la cuenta
    -Usuario de la app con posibilidad de registrar gastos e ingresos
-Gestion de Gastos
    -Tipo
    -Moneda
    -Descripcion
    -Gráfica de gastos por:
        -tipo
        -moneda
        -agrupados
